**正常な値**: Re(ε) ≈ 12（静電誘電率）、Im(ε) ≈ 0（バンドギャップ以下）

### 原因特定

```bash
# GS計算（sa001）の入力ファイルを確認
cat ~/SALMON/calc/sa001/Si_gs.inp
# → num_rgrid = 16, 16, 16
# → num_kgrid = 16, 16, 16

# Exercise-5の入力ファイルを確認
cat ~/SALMON/calc/ex05/Si_rt_response.inp
# → num_rgrid = 12, 12, 12
# → num_kgrid = 4, 4, 4
```

**原因: GS計算と RT計算でグリッドが不一致。**  
SALMONの `tddft_response` は restart から GS波動関数を読み込むが、グリッドが違うと波動関数が正しく展開できず、誤った結果になる。

### GS再計算（正しいグリッドで）


> 💡 **教訓**: GS計算は 16×16×16 → 4×4×4 に変更するだけで **2734秒 → 162秒** に短縮（17倍）。

## Exercise-6 結果確認（2026-04-09）

Exercise-6 の計算も既に完了していた（Job 13088）。

### 物理的解釈

**Electric Field**: cos²包絡のパルス（hw = 1.55 eV、I = 10¹² W/cm²）

**Power Spectrum の見方**:
- 1ω（1.55 eV）: 基本波。最も強い。
- 3ω, 5ω, 7ω...: 奇数次高調波（HHG）。強度が指数関数的に減衰。
- 10¹² W/cm² は比較的弱い強度なので、高次（>10ω）は急激に弱まる。
  強度を上げると（~10¹³〜10¹⁴ W/cm²）高次高調波プラトーが現れる。

**Excitation Energy ≈ 1.2 eV/cell**:
- 光子エネルギー 1.55 eV < バンドギャップ 2.4 eV なので**1光子では励起できない**
- 2光子過程（2×1.55 = 3.1 eV > 2.4 eV）による多光子励起が主因
